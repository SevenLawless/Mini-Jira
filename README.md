# Bug Tracking / Mini Jira

A lightweight bug-tracking and project management web app for small teams.

The app helps teams create workspaces, manage projects, track tasks/bugs, move issues through workflow columns, discuss tasks with comments, and view activity history.

## Project Goal

Build a clean, practical MVP inspired by tools like Jira, Linear, and Trello.

The goal is not to clone those tools fully. The goal is to build a useful full-stack project that demonstrates real-world development skills:

- authentication
- roles and permissions
- CRUD operations
- database design
- task workflow
- drag-and-drop board
- comments
- activity logs
- search and filtering
- clean GitHub workflow

## Main Features

- User authentication
- Workspaces
- Workspace members and roles
- Projects
- Board columns
- Tasks / issues
- Drag-and-drop task movement
- Comments
- Activity log
- Search and filters
- Seed/demo data

## MVP Status

Current phase: Planning and project setup.

## Local Development Setup

This project currently runs locally.

Required tools:

- Node.js
- npm
- MySQL Server
- MySQL Workbench
- Git

The database is local for now. Hosting and deployment will be handled later after the MVP is complete.
